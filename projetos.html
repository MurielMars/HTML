Sistema de Automação Residencial

Desenvolver um sistema simples de automação para controle de iluminação e outros aparelhos elétricos de uma casa. Usando microcontroladores como Arduino ou Raspberry Pi, é possível criar um sistema que permita controlar lâmpadas e tomadas via smartphone ou por comandos de voz, integrando sensores de movimento ou de presença.
